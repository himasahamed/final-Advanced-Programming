How to use the system

Follow these simple steps to complete a reservation and generate the bill.

1) Login

Open the Login page and sign in using your username and password. After login, the Home page will show reservation options.

2) Add Reservation

Enter guest details, room type, check-in and check-out dates. The system saves the reservation to MySQL.

3) View Reservation

Enter Reservation No to view the saved reservation details and confirm the booking.

4) Generate Bill

Enter Reservation No to calculate nights and total amount. Use the Print button to print or save as PDF.


Troubleshooting

• If pages do not update, restart Tomcat and hard refresh the browser (Ctrl + F5).
• If DB connection fails, ensure MySQL Server is running and port is correct (3307).
• If login fails, verify admin user exists in the users table.

Tip: Home changes automatically after login using session.
